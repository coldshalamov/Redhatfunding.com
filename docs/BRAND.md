# Brand Tokens

| Token | Value | Usage |
| --- | --- | --- |
| `--brand` | `#cc0000` | Primary buttons, accents, progress gradient |
| `--ink` | `#0b1320` | Headlines, primary text |
| `--muted` | `#5b6471` | Secondary copy, table metadata |
| `--line` | `#e6eaf0` | Borders, dividers |
| `--success` | `#16a34a` | Success messages |
| `--error` | `#b91c1c` | Validation errors |

## Typography

- Headings: **DM Sans**, weight 600–700
- Body copy: **Inter**, weight 400–600
- Letter spacing: tight for headings, normal for body text

## Components

- Border radius: 24px (`rounded-3xl`)
- Shadow: `0 12px 36px rgba(11,19,32,0.08)`
- Focus ring: `outline: 2px solid color-mix(in oklab, var(--brand), white 30%)`
- Toast notifications auto-dismiss after 5 seconds but remain dismissible with a focus-visible ring around the control for keyboard users.

## Imagery & Tone

- Convey speed, confidence, and partnership.
- Use photography or illustrations with warm reds, neutral backgrounds, and subtle gradients.
