# RedHat Funding

A full-stack experience for RedHat Funding featuring a responsive marketing site, multi-step funding wizard, and administrative lead management.

## Directories

- `frontend/` — React + Vite + TypeScript + TailwindCSS
- `backend/` — FastAPI + SQLAlchemy + Alembic
- `docs/` — Developer documentation (setup, API, brand, deployment)
- `infra/` — Infrastructure references and tooling

See `docs/README.md` for quickstart instructions.
